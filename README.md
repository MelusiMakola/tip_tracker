Tip Tracker 2.0
Tip Tracker 2.0 is a Python-based application designed to streamline tip management for restaurants or cafes. With dynamic features like daily tip entry, weekly summaries, and waiter management, this program helps teams organize and track tips efficiently.

Features:
-Daily Tip Tracking: Record tips for individual waiters with date and time stamps.
-Weekly Summaries: Calculate total tips for each waiter from Monday to Sunday, with a note if the week is incomplete.
-Waiter Management: Add or remove waiters dynamically, ensuring the system adapts to staffing changes.
-Structured Data Storage: All data is stored in JSON files for easy readability and scalability.
-User-Friendly Reporting: View tips in a clean, formatted layout for better insights.

Tech Stack:
-Language: Python 3
-Libraries: json, datetime
-File Storage: JSON files for tips and waiter data

How to Use:
-Clone the repository: git clone https://github.com/MelusiMakola/tip-tracker.git
-Navigate to the project directory: cd tip-tracker
-Run the program: python tip_tracker.py
